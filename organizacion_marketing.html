★ Modelo de generación de demanda · hoy
  • Inbound (foco de este informe). Genera demanda y no solo atiende los leads que entran: los nutre desde el propio CRM hasta que están listos.
  • Atribución multicanal. La demanda se cocina en muchos puntos de contacto, canales y momentos. En el CRM registramos por dónde entró el contacto, pero muchas veces otros canales han colaborado o hecho push (nos vieron o consumieron contenido por otras vías); hasta tener el contacto, esa influencia no queda "certificada".
  • Outbound y nuevas vías. Generación de demanda nueva por otros caminos (prospección, Brain, partnerships) — fuera del alcance de este informe.
Marketing es el motor de demanda; Ventas cualifica y convierte; la clave común a todos: aprender y optimizar.

02

Recomendaciones

A partir de los datos de canal, las reuniones y el CRM — priorizadas de mayor a menor impacto.

  1. Calidad > volumen en paid. Google Ads es el que más SQL trae (61) pero el que menos convierte a oportunidad (3%). Afinar fit+hit y creatividades para que el SQL de pago llegue con intención real, vigilando el CPL para no disparar el gasto.
  2. Iluminar el "tráfico directo". Es la mayor fuente de oportunidades inbound (4) y convierte al 14%, pero "directo" esconde el origen real. Cerrar la instrumentación para saber qué hay detrás y poder replicarlo.
  3. Escalar eventos, revisar webinars. Eventos/Campañas convierte 2/2 (señal fuerte pese a base baja) → palanca de Marisa para escalar. El webinar acumula muchos MQL pero 2 SQL y 0 oportunidades → revisar cualificación y seguimiento post-webinar.
  4. Encender los canales ciegos. Referencias de IA, newsletters externas y medios no se miden hoy → instrumentarlos para atribuir sus SQL y oportunidades (trabajo del sistema de septiembre).
  5. Arreglar el nurturing. El primer email va al 45% de apertura, pero el 2º y 3º caen mucho → reescribirlos para mejorar el paso lead → MQL → SQL en todos los canales.
  6. Cerrar el bucle con Ventas. Recoger de forma sistemática por qué las oportunidades no avanzan y volcarlo a scoring y campañas — aprendizaje común.
  7. Higiene de datos continua. Limpiar oportunidades sin negocio asociado y contactos freemium para que el reporting sea fiable.
